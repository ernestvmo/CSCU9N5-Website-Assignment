Creating a Thread

In Java, there are two ways of creating a Thread.

  • Extending the Thread class.

class JavaThread extends Thread
{

...

}


  • Implementing the Runnable interface.

class JavaThread implements Runnable
{

...

}


In general, it is easier to extends the Thread class. But sometimes, we class is already extending some other class, and Java does not allow multiple inheritance.
That is why we have to implement the Runnable interface.

As it is more commonly used, we will be using the first techniques for these Tutorial, unless mentioned.
To create Threads, use the following:

JavaThread thread = new JavaThread();

Starting a Thread

There are three important methods used to control Threads:

  • public void start()

  • public void run()

  • public void stop()


We will cover the first two in this section, and the last one in the next section.


The

start()

method prepares a thread to be run; the

run()

method actually performs the work of the thread - it serves as the main routine for the thread.



The class that extends Thread must redefine the

run()

method of Thread.


⚠

To start the thread actually running, you do not invoke the

run()

method. Rather, you invoke the

start()

method on your object:


thread.start();

The

run()

method is caaled automatically at runtime as necessary, once you have called

start()

.

Stopping a Thread


The thread dies when the

run()

method terminates or when the thread's

stop()

method is invoked:


thread.stop();

It is not recommended to stop a thread using the

stop()

method (it is deprecated, as it is not thread-safe (we will cover this part in the future)).


Naming a Thread

If you create multiple threads from the same class, it is often useful to give them names to tell them apart.

The Thread class provides a constructor to do this:

public Thread(String name) { ... }

This is called using your subclass like this:

class JavaThread extends Thread
{

public JavaThread(String name)
{

super(name);

}


public void run()
{

System.out.println(this.getName());
...

}

}

So when you create your threads:

JavaThread thread1 = new JavaThread("Thread 1");
JavaThread thread2 = new JavaThread("Thread 2");

thread1.start();
thread2.start();

The console will display the name of the two threads created.
