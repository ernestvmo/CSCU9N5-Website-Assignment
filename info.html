Multithreading (using Threads) is a very common and practical function, used in many ordinary programs.

What is a thread?

A Thread is an independent path of execution within a program.
All programs have at least one thread, known as the main Thread (created by the JVM, Java Virtual Machine).

So then, what is multi-threading?

To put it simply, multi-threading is the process of executing multiple threads simultaneously.
